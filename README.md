# Solver for domino puzzle

A C++ application that solves a domino puzzle where some numbers on the board have been erased. The program uses a backtracking algorithm to restore missing values and find the unique placement for a standard set of 28 dominoes.

### Requirements
* A C++ compiler
* Standard C++11 or higher (for `<chrono>` and `auto` support)

### Compilation
```bash
g++ task15.cc -o task15
```

### Run the program
```bash
./task15
```
